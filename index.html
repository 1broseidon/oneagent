View on GitHub

Normalized Output

Every backend returns the same JSON shape. Stream events, parse results, switch models — your code never changes.

Portable Threads

Start a conversation on Claude, continue on Codex, summarize on Pi. Thread history follows across backends with file locking for concurrency.

Zero-Code Backends

Adding a new agent is a JSON config edit. Define the CLI command, map the output fields, done.

Agent-as-Tool

Dispatch work from one agent to another. The outer agent calls oa, gets normalized results back, chains follow-ups across models.
